Food Point
========================================
## Authors
 - Brian Wang
 - Matt O'hagan
 - Diva Hurtado

## Description
Application to connect farmers and food desert consumers

Collects data of where people need food based on Google Map Consumer Services

Utilizes FireBase, Google Maps, Objective C, XCode
